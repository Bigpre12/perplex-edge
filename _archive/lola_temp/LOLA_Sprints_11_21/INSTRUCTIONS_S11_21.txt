LOLA Sprints 11-21 Integration Guide

SPRINT 11 — Kelly + CLV
  cp sprint11_kelly_router.py backend/routers/kelly_router.py
  cp sprint11_clv_service.py backend/services/clv_service.py
  Add 'clv' and 'clv_label' Float/String columns to BetLog model
  Register: app.include_router(kelly_router)
  Frontend: Show recommended_units on every prop card from GET /api/kelly/size/{prop_id}

SPRINT 12 — SGP Builder
  cp sprint12_sgp_router.py backend/routers/sgp_router.py
  Register: app.include_router(sgp_router)
  Frontend: Add 'SGP Mode' toggle in ParlayBuilder.tsx that POSTs to /api/sgp/build
  GET /api/sgp/correlation-matrix?sport=NBA returns matrix for heat map display

SPRINT 13 — Hedge Calculator
  cp sprint13_hedge_calculator.py backend/routers/hedge_router.py
  Register: app.include_router(hedge_router)
  Frontend: Add Hedge Calculator modal in BetTracker — input original stake/odds + hedge odds

SPRINT 14 — PWA
  cp sprint14_manifest.json public/manifest.json
  cp sprint14_service_worker.js public/sw.js
  cp sprint14_pwa_setup.tsx components/PWASetup.tsx
  Add <PWASetup /> to app/layout.tsx
  Add manifest link tags to <head> (see comments in PWASetup.tsx)
  Generate icons: 192x192 and 512x512 PNG -> public/icons/

SPRINT 15 — H2H + B2B Flags
  cp sprint15_h2h_service.py backend/services/h2h_service.py
  Requires PlayerStats model to have: opponent_team, is_home fields
  Call check_back_to_back() in your prop sync and store fatigue_flag on PropLine
  Display B2B badge in prop card UI

SPRINT 16 — Weather
  cp sprint16_weather_service.py backend/services/weather_service.py
  Add OPENWEATHER_API_KEY to .env (free tier at openweathermap.org)
  Call get_game_weather(team_abbr) in your daily sync for NFL/MLB games

SPRINT 17 — Referee Data
  cp sprint17_referee_service.py backend/services/referee_service.py
  Requires RefereeGame model: ref_name, total_fouls, pace, total_points, fta
  Requires Schedule model to have: referee_crew (comma-separated names)
  Data source: NBA.com game officials endpoint or manual import

SPRINT 18 — Contests
  cp sprint18_contest_router.py backend/routers/contest_router.py
  Requires Contest + ContestEntry models (see new models below)
  Register: app.include_router(contest_router)
  Create contests manually in DB or build admin endpoint

SPRINT 19 — Discord Bot
  cp sprint19_discord_bot.py backend/services/discord_bot.py
  Add DISCORD_WEBHOOK_URL to .env (Settings > Integrations in Discord)
  Add to startup in main.py:
    from services.discord_bot import daily_digest_loop
    asyncio.create_task(daily_digest_loop())

SPRINT 20 — White-Label API
  cp sprint20_api_tier_router.py backend/routers/api_tier_router.py
  Requires APIKey model: key, tier, is_active, expires_at, rate_limit_per_hour, requests_this_hour, total_requests, last_used, last_reset
  Register: app.include_router(api_tier_router)
  Generate keys via admin endpoint or DB insert

SPRINT 21 — SharpMush Signal
  cp sprint21_sharpmush_service.py backend/services/sharpmush_service.py
  Call get_smart_money_signal(db) from a /api/smart-money endpoint
  Display on dashboard as 'LOLA Smart Money' section

NEW .ENV VARIABLES NEEDED:
  OPENWEATHER_API_KEY=your_key (free at openweathermap.org)
  DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/...
